# 작업 맥락 기록

## 2026-07-12 스크롤 고정 경계 변경

- 첨부 이미지의 빨간 선을 실제 스크롤 경계로 사용한다.
- 랭킹은 필터 및 옵션 아래의 순위 목록만 스크롤한다.
- 선수 기록은 필터 및 옵션을 고정하고 표의 열 제목 아래 데이터 행만 스크롤한다.
- 동적 필터 높이를 계산하지 않고 모바일 페이지를 세로 flex 레이아웃으로 나눠 남은 공간만 스크롤한다.
- 내부 목록이 위로 스크롤된 상태에서는 당겨서 새로고침이 시작되지 않게 한다.

## 2026-07-12 필터 폭 통일

- 모바일의 랭킹 항목과 전체 시즌 선택은 지역 선택 필드와 같은 3분의 1 폭으로 통일한다.
- 데스크톱의 전체 시즌 선택은 지역 선택과 같은 200px 폭으로 통일한다.
